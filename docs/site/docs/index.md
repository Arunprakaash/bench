# Bench — Developer Documentation

Bench is an agent testing platform. Define multi-turn conversation scenarios, run them against your agent, and get turn-by-turn pass/fail results with LLM-based evaluation.

## What can you do with Bench?

- **Test any agent** — connect local Python agents or any external HTTP service
- **Write multi-turn scenarios** — simulate real conversations with expected outcomes per turn
- **Evaluate with LLM judgment** — intent-based assertions evaluated by an LLM judge, not brittle string matching
- **Track regressions** — schedule runs and get alerted when passing scenarios start failing
- **Integrate with CI** — run your test suite on every deploy using the REST API
